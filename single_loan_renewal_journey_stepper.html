Step-by-step journey of one loan renewal through the LOS and agent platform, showing which steps are handled by the system, agents, or humans.

1 · The tickler fires

A nightly job scans maturities and finds this line 90 days out. The LOS writes a renewal.due event, opens a renewal case, and sets status to documents requested. Nothing intelligent has happened yet — just the system of record keeping its calendar.

2 · Intake agent works the checklist

The event wakes the intake agent. It builds the document checklist (new fiscal-year statements, updated debt schedule, guarantor PFS), drafts the request email for the RM, and as PDFs arrive it classifies each one, ticks the list, and flags gaps. Writes back: document statuses, checklist progress.

3 · Spreading agent extracts and maps

Documents complete → docs.complete event → spreading agent extracts the financials, maps them to the standard spread, computes draft DSCR, leverage, and global cash flow — each number carrying a confidence score and a source-page reference. Then it stops, on purpose.

4 · Underwriter gate — the interrupt

The LangGraph run pauses and checkpoints. The underwriter sees the draft spread beside the source pages, corrects two mappings, approves. The run resumes with corrections in state. The spread of record is now human-approved — nothing downstream consumed it before this moment.

5 · Rules, compliance branch, and the EWS score

The deterministic layer runs: rules engine tests policy (DSCR slipped to 1.15x vs the 1.20x minimum → waivable exception, citation §4.3.1); the compliance sub-workflow routes it to credit-manager authority; XGBoost scores deterioration risk at 0.71 — this renewal jumps the queue.

6 · Renewal agent drafts the review

With verified facts, exceptions, last year's review (retrieved via RAG), and the EWS reasons, the agent drafts the annual review: deltas narrated, covenant status, the exception narrative with its citation, proposed terms. Facts-only contract; the recommendation field is left for a human.

7 · Approver gate, then the LOS closes the loop

Second interrupt: the credit manager reads the draft, signs the DSCR waiver, approves renewal at reduced limit. The decision, signature, and full audit trail write back to the LOS; status flips to renewed; monitoring resumes with next year's tickler already scheduled. ↻
